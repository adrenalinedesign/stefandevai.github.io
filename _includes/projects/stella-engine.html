Stella Engine:

A cross-platform 2D C++/OpenGL game engine. It's currently in its early development phase.

Current features:

  • Batch Rendering
  • Spritesheets
  • Animations
  • Particles
  • Framebuffers
  • Bitmap font rendering using awesome usr_share Grafx2 fonts
  • Responsive audio streaming
  • Basic 2D AABB physics
